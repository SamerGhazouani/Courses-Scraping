# Course Scraping Automation

This project is an automated system for collecting online course data from multiple educational platforms. It extracts, cleans, and structures course information to create a reliable dataset for a tutorial search platform.

# Features

- Automated website navigation and content extraction using Selenium

- Fast HTML parsing with BeautifulSoup

- Extraction of detailed course data: title, level, description, skills, instructor, duration, etc.

- Data cleaning and validation to remove duplicates and inconsistencies

- Integration of cleaned data into a structured database

- Scheduled updates to keep the course catalog current
